---

Send a password reset email to the user's registered address.

## Request Body

<ParamField body="email" type="string" required>
  User's email address
</ParamField>

## Responses

<ResponseField status="200" name="Password reset email sent">
  Password reset initiation confirmation
</ResponseField>

<ResponseField status="400" name="Bad Request">
  Invalid input or missing fields
</ResponseField>

<ResponseField status="500" name="Internal Server Error">
  An unexpected error occurred
</ResponseField>
