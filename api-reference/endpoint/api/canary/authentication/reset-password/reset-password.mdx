---

Reset a user's password using a reset token and new password.

## Request Body

<ParamField body="token" type="string" required>
  Password reset token provided in the password reset email
</ParamField>

<ParamField body="password" type="string" required>
  New password
</ParamField>

## Responses

<ResponseField status="200" name="Password reset successfully">
  Password reset confirmation
</ResponseField>

<ResponseField status="400" name="Bad Request">
  Invalid token or missing fields
</ResponseField>

<ResponseField status="500" name="Internal Server Error">
  An unexpected error occurred
</ResponseField>
