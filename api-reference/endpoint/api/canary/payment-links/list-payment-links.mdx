Retrieve all payment links created by the current authenticated merchant.

## Query Parameters

<ParamField query="page" type="number">
  Page number
</ParamField>

<ParamField query="limit" type="number">
  Records per page
</ParamField>

## Response

<ResponseField status="200" name="List Payment Links">
  <ResponseField name="data" type="array">Array of payment link objects</ResponseField>
  <ResponseField name="pagination" type="object">Pagination data</ResponseField>
</ResponseField>

## Authentication

This endpoint requires Bearer token authentication.
