## Query Parameters

<ParamField query="page" type="number">
  Page number
</ParamField>

<ParamField query="limit" type="number">
  Records per page
</ParamField>

## Response

<ResponseField status="200" name="List Payouts">
  <ResponseField name="data" type="array">Array of payout objects</ResponseField>
  <ResponseField name="pagination" type="object">Pagination data</ResponseField>
</ResponseField>

This endpoint requires Bearer token authentication.
