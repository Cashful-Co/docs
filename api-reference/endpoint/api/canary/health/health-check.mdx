---
title: "Health Check"
description: "Performs a health check on the application and external dependencies"
api: "GET /api/canary/health"
---

Performs a health check on the application and external dependencies.

## Response

<ResponseField status="200" name="Healthy">
  <ResponseField name="info" type="object">
    High-level health info about services
    <Expandable title="Info Object">
      <ResponseField name="cashful-api" type="object">Service health for the API</ResponseField>
      <ResponseField name="database" type="object">Service health for the database</ResponseField>
    </Expandable>
  </ResponseField>
  <ResponseField name="error" type="object">Any errors returned by the health check (empty object when healthy)</ResponseField>
  <ResponseField name="details" type="object">Detailed status of individual dependencies
    <Expandable title="Details Object">
      <ResponseField name="cashful-api" type="object">Status for the API service</ResponseField>
      <ResponseField name="database" type="object">Status for the database service, may include a message when down</ResponseField>
    </Expandable>
  </ResponseField>
</ResponseField>

## Example Response (Healthy)

```json
{
  "info": {
    "cashful-api": { "status": "up" },
    "database": { "status": "up" }
  },
  "error": {},
  "details": {
    "cashful-api": { "status": "up" },
    "database": { "status": "up" }
  }
}
```

## Example Response (Unhealthy)

```json
{
  "info": {
    "cashful-api": { "status": "up" },
    "database": { "status": "down" }
  },
  "error": {
    "database": {
      "status": "down",
      "message": "Database connection failed"
    }
  },
  "details": {
    "cashful-api": { "status": "up" },
    "database": { "status": "down", "message": "Database connection failed" }
  }
}
```

## Error Responses

<ResponseField status="503" name="Service Unavailable">
  One or more health checks failed. The application or its dependencies are not healthy.
</ResponseField>

## Authentication

This endpoint does not require authentication and can be called by any client.
